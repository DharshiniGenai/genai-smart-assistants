AI Travel Planner ✈️🌍
>🚀 Built as a real-world AI travel assistant using Groq API + LLaMA models

An intelligent AI-powered travel itinerary generator that creates personalized, structured, and budget-friendly 3-day travel plans using Groq API (LLaMA models).

📖 Project Overview

The AI Travel Planner is a beginner-friendly Python application that helps users generate complete travel itineraries based on destination, budget, and interests using Artificial Intelligence.

Instead of manually searching for places, hotels, food options, and plans, users simply enter basic travel details. The AI system then generates a complete 3-day travel itinerary including sightseeing plans, food suggestions, travel tips, and estimated expenses.

This project demonstrates how Generative AI can be used to build smart travel planning systems that are practical, personalized, and easy to use.

The application uses the Groq API with LLaMA models to generate realistic and well-structured travel plans.

⚙️ How the Project Works

The program first asks the user for travel-related information such as destination city, budget, and interests.

After collecting the inputs, the application creates a structured AI prompt and sends it to the Groq API using LLaMA models.

The AI model analyzes the user’s travel preferences and generates a complete 3-day itinerary including places to visit, food recommendations, travel tips, and daily schedules.

Finally, the generated travel plan is displayed directly in the terminal in a clean and structured format.

📦 Installation

First, install the required packages:

pip install groq python-dotenv
🔑 API Key Setup

To use this project, you need a Groq API key.

Get your API key from:
https://console.groq.com/keys

After generating the API key, create a .env file and add:

GROQ_API_KEY=your_api_key_here

Then load it in your code:

from dotenv import load_dotenv
import os

load_dotenv()
api_key = os.getenv("GROQ_API_KEY")
▶️ Running the Project

Run the Python file using:

python travel_planner.py

The application will ask for travel details and generate a personalized itinerary automatically.
